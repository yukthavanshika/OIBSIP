## Objective

The objective of this task is to perform basic network scanning using Nmap and identify open ports and active services running on a system.

---

## Tool Used

- Nmap (Network Mapper)

---

## Steps Performed

### 1. Installed Nmap

Nmap was installed successfully on the system.

The installation was verified using:

nmap --version


### 2. Performed Network Scan

A scan was performed on the local machine using:

nmap localhost


### 3. Analyzed Results

The scan detected open ports and services running on the system.

Detected services:

- 3306/tcp - MySQL
- 5000/tcp - UPnP
- 7000/tcp - afs3-fileserver

## Importance of Network Scanning

Network scanning helps cybersecurity professionals identify active devices, open ports, and running services.

This information helps detect possible vulnerabilities and improve system security.
